# Swelter Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A live-input tape + heat-haze effect for monome norns: external audio passes through a six-lens chain (saturation, refraction, mirage, heat-haze, dropout, tape-age), performed from three encoders with page cycling and a layered haze-field screen.

**Architecture:** One `CroneEngine` (`lib/Engine_Swelter.sc`) hosts a single stereo `SynthDef` that reads the audio input and writes a dry/wet blend to the engine's own output bus. All six lenses are a fixed serial chain inside that one SynthDef; each is one labeled section. `swelter.lua` defines params (`set_action` → `engine.*`), mutes the hardware input monitor so the engine is the sole input path, drives the page-cycled encoder UI, and renders the layered screen from two polls (`amp_in`, `haze_mod`).

**Tech Stack:** SuperCollider (sclang/scsynth via norns Crone), Lua (norns API: `params`, `poll`, `metro`, `screen`). Spec: `docs/superpowers/specs/2026-05-24-swelter-design.md`.

**Verification reality:** norns DSP cannot be unit-tested off-device. `sclang` is **not** installed locally, so `.sc` tasks verify by structural self-review against this plan and an on-device load. Lua tasks verify with `luac -p`. Final task is an on-device playtest checklist the user runs. Per the user's workflow, SynthDef changes require **SYSTEM > RESTART** on the device; never restart norns-sclang over SSH.

### Task 2: Engine scaffold (CroneEngine, full SynthDef with identity lenses, all commands + polls)

**Files:**
- Create: `lib/Engine_Swelter.sc`

This task establishes the full arg list, command set, and poll set. Each lens is a clearly marked identity section (`sig` unchanged) that later tasks replace. After this task the engine is a dry/wet passthrough with working level + output controls.

- [ ] **Step 1: Write the full engine scaffold**

```supercollider
// Engine_Swelter
//
// Live-input tape + heat-haze effect. One stereo SynthDef reads the audio
// input and writes a dry/wet blend to the engine output bus. Six lenses form
// a fixed serial chain; each is one labeled section.
//
// commands (all "f"): trim drywet out_trim drive
//   wow wow_rate flutter flutter_rate mirage mirage_detune
//   haze haze_rate haze_turb dropout age hiss
//
// polls: amp_in (input amplitude 0..1), haze_mod (modulation amount 0..1)

Engine_Swelter : CroneEngine {
  var synth;
  var ampBus, hazeBus;

  *new { arg context, doneCallback;
    ^super.new(context, doneCallback);
  }

  alloc {
    ampBus  = Bus.control(context.server, 1);
    hazeBus = Bus.control(context.server, 1);

    SynthDef(\swelter, {
      arg in_l, in_r, out, amp_out, haze_out,
          trim=1, drywet=0.5, out_trim=1,
          drive=0,
          wow=0, wow_rate=0.5, flutter=0, flutter_rate=8,
          mirage=0, mirage_detune=8,
          haze=0, haze_rate=2, haze_turb=0.5,
          dropout=0, age=0, hiss=0,
          slew=0.05;

      var dry, sig, hazemod;

      // smoothed controls
      var ktrim  = Lag.kr(trim, slew);
      var kwet   = Lag.kr(drywet, slew);
      var kout   = Lag.kr(out_trim, slew);
      var kdrive = Lag.kr(drive, slew);
      var kwow   = Lag.kr(wow, slew);
      var kflut  = Lag.kr(flutter, slew);
      var kmir   = Lag.kr(mirage, slew);
      var kdet   = Lag.kr(mirage_detune, slew);
      var khaze  = Lag.kr(haze, slew);
      var kturb  = Lag.kr(haze_turb, slew);
      var kdrop  = Lag.kr(dropout, slew);
      var kage   = Lag.kr(age, slew);
      var khiss  = Lag.kr(hiss, slew);

      dry = In.ar([in_l, in_r]) * ktrim;
      sig = dry;
      hazemod = DC.kr(0);

      // === SATURATION ===        (identity — filled in Task 3)
      // === REFRACTION ===        (identity — filled in Task 4)
      // === MIRAGE ===            (identity — filled in Task 5)
      // === HEAT-HAZE ===         (identity — filled in Task 6)
      // === DROPOUT ===           (identity — filled in Task 7)
      // === TAPE-AGE ===          (identity — filled in Task 8)

      // dry/wet crossfade (-1 = dry, +1 = wet) + output trim
      sig = XFade2.ar(dry, sig, (kwet * 2) - 1) * kout;

      // polls
      Out.kr(amp_out, Amplitude.kr((dry[0] + dry[1]) * 0.5, 0.05, 0.2));
      Out.kr(haze_out, Lag.kr(hazemod.clip(0, 1), 0.1));

      Out.ar(out, sig);
    }).add;

    context.server.sync;

    synth = Synth.new(\swelter, [
      \in_l,     context.in_b[0].index,
      \in_r,     context.in_b[1].index,
      \out,      context.out_b.index,
      \amp_out,  ampBus.index,
      \haze_out, hazeBus.index
    ], context.xg);

    // commands — one per controllable arg
    [\trim, \drywet, \out_trim, \drive,
     \wow, \wow_rate, \flutter, \flutter_rate,
     \mirage, \mirage_detune,
     \haze, \haze_rate, \haze_turb,
     \dropout, \age, \hiss].do { |name|
      this.addCommand(name, "f", { |msg| synth.set(name, msg[1]) });
    };

    // polls
    this.addPoll(\amp_in,   { ampBus.getSynchronous });
    this.addPoll(\haze_mod, { hazeBus.getSynchronous });
  }

  free {
    synth.free;
    ampBus.free;
    hazeBus.free;
  }
}
```

function enc(n, d)
  activity = util.time()
  if n == 1 then
    params:delta("drywet", d)
  else
    local p = PAGES[page]
    local id
    if n == 2 then id = shift and p.e2s or p.e2
    else            id = shift and p.e3s or p.e3 end
    if id then params:delta(id, d) end
  end
  redraw()
end

function key(n, z)
  activity = util.time()
  if n == 1 then
    shift = (z == 1)
  elseif n == 2 then
    if z == 1 then page = (page % #PAGES) + 1 end
  elseif n == 3 then
    bypass = (z == 1)
    engine.drywet(bypass and 0 or params:get("drywet"))
  end
  redraw()
end
